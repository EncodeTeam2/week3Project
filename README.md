# Hello, tokens!

Learning ERC20 and ERC721 EIP standards through a token sale.

Working with:

- OpenZeppelin contracts

## Dependencies

Install open zeppelin contracts

```
yarn add @openzeppelin/contracts --dev
```

## Test

```
yarn hardhat test
```

## Setup

Put your .env file at root. It should contain:

```
PRIVATE_KEY=
INFURA_API_KEY=
INFURA_API_SECRET=
ALCHEMY_API_KEY=
ETHERSCAN_API_KEY=
```
